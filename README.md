## What is this?
Job Stack is an enlisting system that is beneficial for job seekers and companies looking for employees. It is developed using Java and Google Firebase. Cloud Firestore in Google Firebase deals with managing the database in this application. Java is used for managing the outer view of the application. This application has a login portal for users. Users will enter their personal as well as professional details while registering themselves into the system. The companies can register themselves by entering their details like Company name, email id, and office address. Users or companies can post their new job openings by providing details regarding the Company name, Job title, Number of vacancies, Job description, Job profile, Criteria, and Position type. The users can view and apply for the jobs provided by the companies or individuals. After applying, The company or individual can view the 
list of user profiles who have applied to a particular job. A chat system is implemented for exchanging information between users. The Admin of the system 
has the access to all the portals. He handles all the login credentials and databases stored in the cloud. The Admin has the main authority who can do insertion, deletion, and modification of data if required.

## Features

#### Login
Both the user and companies can log in to the system using their registered email and password.
#### Sign Up
To enter the system user and company should register first. Requirements of registration are first name/company name, last name, email id, password, and phone number.
#### Forgot Password
If the user forgot the password, the user can send a reset link to the mail id to reset the password.
#### Logout
The user can log out from the system after their use. 
#### Add Jobs
Companies can post jobs opening in their account to hire people.
#### Search Jobs
Users can search jobs according to their appropriate industry.
#### Filter 
The user can filter the job according to their appropriate categories.
#### Contact Companies
Users can contact the company using the email or phone number or the registration link provided by the company.
#### Chat option
This option provides users and companies to interact with each other.
#### View and edit profile
Users and companies can view and edit their profiles as per their needs.
#### Change Password
After logging into the system, the users or companies can change their existing password by entering their old password and the new password.
